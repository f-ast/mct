This directory contains a TXL base grammar and pretty-printer for Javascript.
Intended for use in a comprehensive HTML web page parser, but can be used alone.
Validated to conform to ECMAscript standard ECMA-262, 5th Edition, Dec 2009

Examples:
	txl Examples/hpc.js
