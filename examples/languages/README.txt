TXL Language Grammars
---------------------

These examples represent a few of the different kinds of
languages that have been processed using TXL.  

These languages are only a tiny subset of the set of 
languages that have been transformed using TXL.

For a complete set of up-to-date language grammars, 
see the TXL World web page,

	http://www.txl.ca/txlworld.html
