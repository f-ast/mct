TXL Example Specification
-------------------------

Name:        
        Cpp.Grm
        CppCommentOverrides.Grm
        cpp.Txl

Description:
        TXL basis grammar and pretty printer for ANSI C++.
        Intended as a basis for creating TXL transformations of
        C++ programs.  Can be used with or without preserving of
        comments.

Author:
        J.R. Cordy & M. Elges

Affiliation:
        Queen's University, Kingston, Canada
        Advanced Software Technologies Inc., Boulder, Colorado, USA

Date:
        February 1992, revised March 1995, revised January 2001

Example:
        txl -comment groff.cpp
