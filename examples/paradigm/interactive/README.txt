TXL Example Specification
-------------------------

Name:        
        io.Txl

Description:
        Demonstration of the use of the external functions
        [get], [getp], [put], [putp], [parse], [unparse],
        [index] and [:] to create an interactive interface with
        interesting string and item manipulation.
